# Box Plot Analysis App

## Description
The VB.NET Data Analysis Application is a versatile tool that allows users to import and 
analyze CSV datasets. With a user-friendly interface, the application provides a seamless 
experience for data manipulation and exploration. This project is built using the VB.NET 
programming language and utilizes various functionalities to facilitate efficient data 
analysis.

## Installation
To install and run the application, follow these steps:

1. Download the zip file from the GitHub repository.
2. Run the setup.exe file.
3. Follow the onscreen instructions to complete the installation process.

## Usage
To use the application, follow these steps:

1. After installation, run the application.
2. For testing purposes, import the test.csv dataset included in the repository.
4. Proceed through all the steps involved in the application.

## Requirements
The application has the following requirements:
Hardware Requirements:
- Processor: 1 GHz or faster processor
- RAM: 2 GB or more

- Display: Minimum screen resolution of 1024x768 pixels
Software Requirements:
